# AgriVerse Episode 1 — CC0 Audio Bundle

This bundle contains four processed 48 kHz stereo ambience loops and three
processed 48 kHz mono interaction effects, along with the original downloaded
CC0 files and saved source pages.

## Final files

- `Audio/Ambience/Canal_Loop.ogg`
- `Audio/Ambience/Wind_Loop.ogg`
- `Audio/Ambience/Birds_Loop.ogg`
- `Audio/Ambience/Insects_Loop.ogg`
- `Audio/SFX/Water/WaterScoop_01.wav`
- `Audio/SFX/Water/VialHandle_01.wav`
- `Audio/SFX/Water/VialCap_01.wav`

The ambience recordings are representative sound design sources rather than
recordings made in Vietnam. Each final ambience loop is approximately 89
seconds and uses a one-second end-to-start crossfade. Interaction filenames
describe their intended game use; consult `SOURCE.md` for the original sound.

Before final artistic approval, audition every file in the Unity scene at the
intended mix level. The source-page descriptions contain no voices, music, or
engines, but listening remains part of Human Gate B.
