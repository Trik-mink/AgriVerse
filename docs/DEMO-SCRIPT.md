# AgriVerse demo script

Target duration: **2:42**. Every game shot below is a real Unity capture. Small title cards,
chapter labels, and captions may be added as editorial overlays; they must not imply features
or facts absent from the game.

Core message:

> Students investigate evidence, negotiate perspectives, model consequences, and revise.
> They do not merely chat with AI.

## Codex and GPT-5.6 caption

Use one restrained editorial card around 2:25:

> Codex accelerated the scenario engine, Express/OpenAI boundary, Unity implementation,
> structured contracts, tests, asset integration, and release debugging. The human set the
> learning design, factual boundaries, revision mechanic, and visual direction. GPT-5.6
> remains the live engine for stakeholders, consequences, grounded feedback, and the policy
> brief.

Do not describe deterministic UI fixtures as live model calls. If an edited visual retake
uses recorded contract-valid responses, label it **Recorded presentation replay**. Retain at
least one clearly labeled **Live hosted GPT-5.6 journey** excerpt.
